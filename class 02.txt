네트워크 프로그래밍
소켓 기반 프로그래밍, 따라서 소켓 프로그래밍이라고도 함
네트워크 연결로 컴퓨터 사이의 데이터 송수신 프로그램 작성

소켓
네트워크 연결 도구
운영체제에 의해 제공되는 소프트웨어적 장치
(프로그래머로 하여금 물리적, 소프트웨어적 상세 내용을 신경쓰지 않게 함)

소켓은 소켓 함수의 호출을 통해 생성함
단 송신과 수신의 소켓 생성 방법에는 차이가 있음

소켓에도 주소정보가 할당 됨 == ip / port 번호
매 연결마다 각각의 소켓이 필요함


서버 측
socket() == 생성 함수
bind() == 주소할당 함수
listen() == 연결 가능 상태로 변경 함수
listen 이후 -> accept() == 연결 요청에 대한 수락 함수, 이 이후 양방향 통신이 가능

이런 listen을 진행하는 소켓을 서버 소켓 혹은 리스닝 소켓이라 지칭


클라이언트 측
socket() == 생성 함수
connect() == 연결 요청 함수


리눅스에서 파일과 소켓을 동일하게 처리함
